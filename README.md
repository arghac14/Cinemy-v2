### How to run Cinemy v2.0

1. Run these following commands:
``` 
cd frontend
npm install
npm run serve
```
2. Run the API on local: Open Backend/IMDB.sln and run on IIS.

3. Open SSMS/SQL Server and Run the IMDBStructure.sql file.
